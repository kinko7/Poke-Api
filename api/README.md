Project - Pokemon Api 🐕
 

✔Functionalities: -Get All Pokemons from Api, Filter by Name, by Id and by Types. Delete and Update -Create a new Pokemon

🔹Technologies used: ReactJS -NodeJS- ReduxJS -JavaScript -PosgresSQL -Express -Sequalize  

First you'll have to clone this repository in a folder in your computer, then you'll have to open your console in that directory and:

cd api
npm install
npm start
This will start the back-end server on your localhost:3001/

Aditionally you'll have to create your local SQL database called 'pokemons' in order to store your own dogs.
